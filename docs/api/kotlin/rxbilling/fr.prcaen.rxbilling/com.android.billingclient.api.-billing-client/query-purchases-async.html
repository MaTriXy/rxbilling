queryPurchasesAsync

@CheckReturnValue @NonNull @SchedulerSupport("none") fun BillingClient.queryPurchasesAsync(skuType: String): Single<List<Purchase>>

Get purchases details for all the items bought within your app. This method uses a cache of Google Play Store app without initiating a network request.

Note: queryPurchasesAsync does not operate by default on a particular Scheduler.

Error handling: If Billing result is not BillingResponseCode.OK, the emitter will deliver RetrievePurchasesException.

Parameters

skuType - The type of SKU, either "inapp" or "subs" as in SkuType

See Also

BillingClient.queryPurchases

Return
a Single that emits the list of Purchase.
