querySkusDetailsAsync

@CheckReturnValue @NonNull @SchedulerSupport("none") fun BillingClient.querySkusDetailsAsync(skusList: List<String>, skuType: String): Single<List<SkuDetails>>

Perform a network query to get SKUs details.

Note: querySkusDetailsAsync does not operate by default on a particular Scheduler.

Error handling: If Billing result is not BillingResponseCode.OK, the emitter will deliver QuerySkuDetailsException.

Parameters

skusList - Specify the SKUs that are queried for as published in the Google Developer console.

skuType - Specify the type SkuType of SKUs we are querying for.

a Single that emits the list of SkuDetails.
