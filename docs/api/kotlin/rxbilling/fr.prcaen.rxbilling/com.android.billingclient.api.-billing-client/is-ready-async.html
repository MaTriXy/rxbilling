isReadyAsync

@CheckReturnValue @NonNull @SchedulerSupport("none") fun BillingClient.isReadyAsync(): Single<Boolean>

Checks if the client is currently connected to the service, so that requests to other methods will succeed.

Note: isReadyAsync does not operate by default on a particular Scheduler.

Warning: This method should be executed on MainThread, it will throw IllegalStateException if not.

See Also

BillingClient.isReady

Return
a Single that emits true if the client is currently connected to the service, false otherwise.
